How to use the TIDP Creator/Uploader tool

Step 1: Creating the TIDP Options

To start this process you need to create the options that will go into the TIDP template, to do this first select 'I need a Template generated'.

Then a second window will appear below, in that please select the project you want to create the options for.

Once the project is selected click the 'Generate TIDP Template', this will then extract the projects Naming Standard values, List of Templates and upload locations to generate an options list to input into the 'Dropdown Options' tab in the TIDP Template.

The files will then automatically download into your default download location on your PC, this is normally your Downloads folder. Two files will be downloaded one being the TIDP Template and the other the TIDP Template Options.

Step 2: Adding in the Options in the TIDP Template

After you have the two files from step 1 open them up.

Navigate to the Options spreadsheet, then in the TIDP_Options sheet select all (using ctrl + A) and copy to the clipboard.

Once copied naviagate to the TIDP Template and select the 'Dropdown list' sheet, select cell A1 and paste from the clipboard.

This will then populate the dropdowns in the TIDP sheet in the template.

Click image to replay

Step 3: Uploading a populated TIDP

After you have populated your TIDP Template you can upload it here which will then create the placeholder documents in ACC for you to use on the project.

To upload the TIDP select the 'I have a populated Template'.

Then select your project from the dropdown, once selected you can upload your file either by draging and droping into the area or clicking it and selecting your file.

Once the files has been read (displaying the file name and size) click the 'Upload TIDP To ACC' button and once the progress bar is fully blue all your placeholder files will be uploaded to ACC.
